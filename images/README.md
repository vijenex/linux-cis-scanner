# Images Directory

This directory contains screenshots and visual documentation for the Vijenex Linux CIS Scanner.

## Contents

- **CLI Screenshots**: Terminal output examples showing scanner execution
- **Report Screenshots**: HTML and CSV report examples
- **Installation Screenshots**: Installation process documentation
- **Feature Demonstrations**: Visual examples of scanner capabilities

## Usage in Documentation

Images stored here are referenced in the main README.md file to provide visual documentation of:
- Scanner execution and output
- Generated reports and their format
- Installation process
- CLI interface and color-coded results

## File Naming Convention

- `cli-output-*.png` - Command line interface screenshots
- `report-html-*.png` - HTML report screenshots  
- `report-csv-*.png` - CSV report screenshots
- `installation-*.png` - Installation process screenshots
- `feature-*.png` - Feature demonstration screenshots

## Image Requirements

- **Format**: PNG preferred for screenshots
- **Resolution**: High resolution for clarity in documentation
- **Size**: Optimized for web display (< 1MB per image)
- **Content**: Clean, representative examples of actual scanner output
